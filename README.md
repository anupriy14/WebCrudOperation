# 📚 Web Development Tutorials Hub

An interactive, documentation-style web application built in **React** and styled with **Tailwind CSS**. It serves as a structured learning platform covering core concepts of HTML5, CSS3, and Bootstrap, powered dynamically by a local mock database file (`db.js`).

## ✨ Features
* **Topic-Based Navigation:** Clean, organized sections allowing users to easily browse and switch between HTML, CSS, and Bootstrap tutorials.
* **Mock Local Database (`db.js`):** Uses a structured local JavaScript database containing topic objects, descriptions, and code snippets to simulate dynamic data fetching.
* **Responsive Layouts:** A mobile-first, modern documentation interface styled entirely with Tailwind CSS utility classes.
* **Dynamic Content Loading:** Leverages React components to instantly render the correct documentation, code examples, and guides without full page reloads.

## 🛠️ Tech Stack & Concepts
* **Frontend Framework:** React.js for component architecture and reactive UI states.
* **Styling Engine:** Tailwind CSS for a highly responsive, clean, and modern design interface.
* **Data Flow:** Topic definitions, syntax rules, and cheat sheets stored inside a local `db.js` file and rendered efficiently using JavaScript `.map()` and `.filter()` routines.
